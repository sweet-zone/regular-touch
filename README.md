# regular-touch
扩展[Regular](http://regularjs.github.io/)的移动端手势事件

现在只有最简单最常用的
`tap`
`longTap`
`doubleTap`
`swipe`
`swipeLeft`
`swipeRight`
`swipeUp`
`swipeDown`
命名方式和Zepto一致.

使用方法和[DOM事件](http://regularjs.github.io/guide/zh/events/dom.html)一样, 参照[touch.html](https://github.com/zjzhome/regular-touch/blob/master/example/touch.html)
